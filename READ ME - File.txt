Tripistry - Setup Guide

What You Need:

XAMPP installed

Web browser

Quick Setup:
1. Start XAMPP
Open XAMPP Control Panel

Click Start for Apache

Click Start for MySQL

2. Copy Project
Copy the Project folder to: C:\xampp\htdocs\

3. Import Database
Open browser: http://localhost/phpmyadmin

Click New → Name database: practical_5 → Click Create

Click Import → Choose practical_5.sql → Click Go

4. Open Website
Go to: http://localhost/tripistry/login.php
